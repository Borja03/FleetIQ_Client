Getting Started

Welcome to our desktop application! This guide will help you navigate and make the most of our software.

Tip: Familiarize yourself with the different sections of the application to optimize your workflow.

Top Menu

The top menu provides quick access to primary functions of the application.

Menu Options

Go to

  • Paquete: Navigate to the package management section.
  • Envio: Access the shipment management interface.
  • Ruta: View and manage delivery routes.
  • Vehiculo: Manage vehicle-related information.

View

  • Dark mode: Switch the application to dark mode for a better low-light experience.
  • Light mode: Switch the application back to light mode for standard visibility.

Help

  • About Paquete: Learn more about the package management module.
  • About Ruta: Get information regarding the route management functionality.
  • About Envio: Read details about the shipment system.
  • About Vehiculo: Understand vehicle-related features.

User Options

  • Profile: Access and manage your profile settings and password changing.
  • Logout: Log out of the application securely.

Filters

    Use filters to refine and customize your view of the data.

  • Filter By Dates

    • How to use and apply this filter:

    • You can select a date range using two date pickers: From and To.
    • If only the From date is selected and the user clicks Apply, all packages created after the selected date will be displayed.
    • If only the To date is selected and the user clicks Apply, all packages created before the selected date will be displayed.
    • If both From and To dates are selected and the user clicks Apply, all packages created between the two dates will be displayed.
  • Filter By Size: You can filter packages based on size categories:
    • SMALL: Displays only small packages.
    • MEDIUM: Displays only medium-sized packages.
    • LARGE: Displays only large packages.
    • EXTRA_LARGE: Displays only extra-large packages.
  • Filter By Name(Sender/Reciver):As you type the name of the sender or receiver, the table will automatically update to display all matching names that contain the characters you have entered.

Editable Table View

Interact with and modify data directly in the table view.

Editing Instructions

  • How to edit cells: Double-click on a cell to enter edit mode. Once the cell is active, modify its content.
  • Saving changes: Press Enter to confirm and save changes. Clicking outside the cell or pressing Escape will cancel the modification.

Bottom Buttons

Quick actions available at the bottom of the application.

Button Functions

  • Add new Package : Adds a new package with default data that can be modified after.
  • Delete Package :When a row(s) is/are selected, clicking delete will prompt the user for confirmation before removing the package(s).
  • Print Report: Generates a report that can either be saved as a PDF or printed directly.

Context Menu Options
